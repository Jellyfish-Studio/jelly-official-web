## Bug 报告

如果你在使用过程中遇到了问题，为了帮助我们尽快定位和解决，建议您在提交问题时提供以下信息:

**问题描述:**

- 问题的具体表现
- 问题发生的频率(偶尔/每次都会)
- 复现步骤(详细描述如何触发该问题)
- 错误截图或录屏

**环境信息:**

- 是否安装了其他具有划词/剪贴板读写功能的软件（如 PopClip、Maccy、Cherry Studio 等）
- 系统语言设置

**应用日志:**

- 导出方法：在 Jelly 应用内“通用”先打开“显示开发者设置”，然后点击“开发者” -> “导出日志”
