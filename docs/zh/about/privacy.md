# 隐私政策

## 概述

Jelly（以下简称"我们"）尊重并保护用户的隐私。本隐私政策说明了我们如何收集、使用和保护您的个人信息。此隐私政策主要适用于 Jelly 主程序及内部集成的框架和服务，不适用于您自己安装的 Jelly 插件。

## 隐私权限

为了让程序功能正常运行，需要为 Jelly 开启下列隐私权限：

- 辅助功能权限

通过辅助功能权限获取当前选中的文本以及选中文本的位置，用于实现划词助手相关功能。当且仅当您使用到相关功能时， Jelly 才会向您请求该权限。

- 屏幕录制与 OCR 数据

为了提供翻译服务，Jelly 需要获取屏幕录制权限。该权限仅严格用于以下用途：

本地处理： 仅当您主动触发截屏、OCR 或翻译功能时，应用才会访问屏幕数据。

不收集数据： 捕获的屏幕图像仅在您的设备上进行本地处理以提取文本。我们不会将您的屏幕录制内容上传、分享或存储到我们的服务器或任何第三方。

数据留存： 屏幕数据仅在识别过程中暂时存在于内存中。一旦处理完成，数据将从内存中永久删除，除非您明确选择将图像保存到本地，或收藏了该文本条目。

- 剪贴板读写

为了获取您在其他应用中选中的文本，我们需要借助剪贴板。需要剪贴板读取和写入的权限。

通过以上权限所获取到的所有数据，我们均不会保存和分享。

## 数据处理与收集

Jelly 致力于最大程度减少对您数据的收集。我们的数据处理方式分为以下两种情况：

1. 本地及第三方直连模式（默认模式）

在常规翻译、工具调用、OCR 功能中，Jelly 不会收集您产生的核心数据（如翻译文本、识别图片）。

直连服务商：如果您配置了第三方服务商（如 Open AI、DeepL、硅基流动等），Jelly 会将数据直接发送至您配置的服务商服务器。这些数据不经过也不存储于 Jelly 的服务器。

本地密钥：您填入的第三方 API 密钥（Key）仅保存在您的本地设备上，Jelly 不会上传。

2. 统计与分析数据

为了提供更好的使用体验及排查问题，Jelly 可能会收集以下非个人身份信息：

脱敏的使用数据：例如划词翻译的使用频率、功能点击次数等，但绝不会包含您的具体翻译内容。

崩溃数据：为了提高程序的稳定性，我们会收集程序的崩溃堆栈信息。

## 本隐私政策的变更

我们可能会不时地更新我们的隐私政策，以适应 Jelly 最新的情况及新功能的发布。因此，我们建议您定期查看本页面以了解最新的内容。

我们将通过在此页面上发布新的隐私政策来通知您任何更改。所有更改在此页面上发布后立即生效。
