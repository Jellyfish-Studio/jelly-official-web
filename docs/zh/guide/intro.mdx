# 介绍

Jelly 是一款 MacOS 平台的全能翻译助手，拥有数十种翻译相关的功能，可以解决你在工作和学习中遇到的各种难题。其核心设计理念是`「开箱即用」`、`「高度可定制」`。不管你是喜欢折腾的极客，还是怕麻烦的懒人，Jelly 都能满足您的需求。

## 为什么选择 Jelly

### 完全免费、流畅体验

市面上的同类工具的 AI、 OCR 功能往往会收费或限制使用次数。免费工具的使用体验又不太好。Jelly 内置了完全免费的 AI、 OCR 功能，没有任何使用限制。同时，我们极其重视交互设计，为您提供更符合 macOS 现代审美、流畅且优雅的用户界面。

### 拒绝封闭，拥抱 AI 自由

与传统翻译软件仅提供固定服务不同，Jelly 拥有完全开放的 AI 生态。 您可以自由接入 ChatGPT、 Gemini 等顶尖大模型，甚至自定义 Prompt（提示词）来打造专属助手。当然，如果您需要某些特定的服务，也可以通过插件轻松扩展。

### 全局覆盖，原生极速

您可能使用过翻译类浏览器插件。浏览器插件虽好，但无法覆盖所有场景。Jelly 是一款真正的原生桌面端软件。 无论是 PDF、电子书，还是任意软件界面，它都能无处不在。得益于 macOS 原生开发，Jelly 支持极速的本地离线翻译、OCR，在没有网络时依然可靠，且资源占用极低。
