# OCR 服务

## 说明

Jelly 在使用 OCR 划词、单词/句子/段落提取器的功能时，会用到 OCR 服务

OCR 服务暂不支持安装插件。

> 如您对其他 OCR 服务有需求，或者觉得哪家服务效果比较好，欢迎[反馈给开发者](/about/contact)

## 支持服务列表

### 系统 OCR 服务

目前 Jelly 内置仅支持系统离线 OCR 服务（Apple Vision）。

Apple Vision 目前仅在中英混合的时候有比较好的准确率。对于其他语言场景，建议只开启一种语言。（这也是默认 OCR 配置）

Jelly 使用自研的分段算法，对 OCR 结果进行了版面分析后处理。因此在离线环境下也可以使用。若您遇到了分词/分句不准的情况，欢迎[反馈](/about/contact)
