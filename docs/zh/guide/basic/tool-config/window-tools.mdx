# 窗口工具

为了保证“开箱即用”，同时提供最大的自定义灵活性，Jelly 的“窗口工具”配置系统设计得稍微有一些复杂。您可以通过这篇文档来了解如何配置窗口工具。

> 窗口工具的设置界面

## 窗口工具的设计

“窗口工具”是指可以展示在 Jelly 各个 UI 窗口中的一系列文本处理工具。

当您点击“窗口工具”按钮时，Jelly 会尝试以按钮旁边的文字作为输入，执行工具对应的逻辑。

> 如果软件自动获取到了上下文，还会把上下文也传给执行的工具。
>
> “单词气泡窗”和“句子气泡窗”支持上下文的显示，如果当前存在上下文，您可以 hover 输入文本（单词气泡窗）或上下文标记（句子气泡窗）来查看软件自动获取到的上下文。

对于每个“窗口工具”，例如“AI 翻译”、“英汉词典”等等，**_其实例是唯一的_**。当您点击工具后面的小齿轮按钮，对该工具进行配置，或者添加、删除工具时，**该改动将在所有窗口位置生效**。

工具的“可见性”和“列表顺序”是**_每个位置独立的_**。例如，您可以将“英汉词典”工具设置为在单词气泡窗中显示，但在句子气泡窗中不显示。您也可以将“播放音频”工具，在单词气泡窗中设置为列表第一个，而在句子气泡窗中放在后面收进下拉菜单。

目前对于每个位置，最多展示的工具数量是**_固定的_**。超出最大展示数量的工具，会自动收进下拉菜单中。

> 如果您觉得这个数量不合适或需要自定义，欢迎 [反馈](/about/contact)
