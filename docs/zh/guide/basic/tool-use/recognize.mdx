# 识别图中单词/句子/文本行

Jelly 的图像识别是一个比较特色的功能。适用于大段文本阅读，或者连续查看多个单词的释义、发音的场景。

## 使用方法

选中一块屏幕区域后，调用识别图中单词/句子/文本行工具。

在这之后，Jelly 会自动尝试识别出该区域的内容，并且绘制边界框标明。当您使用鼠标 hover 到一个边界框时，根据文本的类型，气泡窗会自动弹出，以便您查看翻译或调用更多工具。

点击屏幕上任意一个识别出的边界框，即可关闭浮层。

## 注意事项

图像识别不会自动刷新。当您识别的内容发生变化时（例如，滚动了屏幕，或者识别的是一个视频），边界框并不会自动更新。

如果您很需要自动刷新的功能，欢迎 [反馈](/about/contact)。

## 上下文控制技巧

图像识别支持手动上下文控制。详见 [控制上下文](/guide/advanced/context-control)
