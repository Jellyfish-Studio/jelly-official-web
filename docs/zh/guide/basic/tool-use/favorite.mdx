# 收藏

Jelly 的收藏功能可能与您常见的应用收藏功能有一些区别。在 Jelly 中，“收藏”不是静态的，它更像一个“历史记录开关”。

**_当你收藏了一个单词，Jelly 会从您的划词、调用工具等操作中自动收集和收藏的单词/句子关联的信息，记录到收藏列表历史中。_**

<MacScreenShot src="/favorite.png" />

因此，您收藏陌生单词后，第二次遇到该陌生单词时，**_建议您主动对包含该单词的句子进行一次划词操作。_** 这样 Jelly 就可以把这句话作为上下文记录了，方便您回想遇到单词的场景。

> Jelly 默认开启自动 OCR 上下文，您也可以对该生词再进行一次划词操作，自动 OCR 获取的文本也会记录到历史中。

## 将收藏标记为已掌握

您可以通过单击收藏列表中的学士帽图标，将收藏的单词/句子标记为“已掌握”。标记为“已掌握”后，Jelly 不再会收集该收藏条目关联的信息、不会再记录上下文。
